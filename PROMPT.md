# PROMPT.md — Session Launcher

Copy and paste this prompt to start each study session:

---

```
Study @GOAL.md for what I'm preparing for and my gaps.
Study @progress.md for my current state.
Study @.agents/specs/README.md for how the learning system works.
Study @.agents/stdlib/README.md for behavioral rules.

Read any files in readings/ that are relevant to my weakest areas.

Based on my progress and gaps, run one focused learning session:
- If this is my first session, start with an interactive quiz on the highest-priority topic from GOAL.md
- If I have prior progress, target my weakest area
- Ask 3-5 questions, one at a time, wait for my answer before moving on
- Grade honestly, push for depth, correct with context

When the session ends, update progress.md with scores and findings.
Tell me what to focus on next.
```
